# Auto-Nest-Car-Rental-Backend

## Project Overview
Auto Nest Car Rental website allows users to browse, book, and rent cars with a simple, user-friendly interface.

## Environment Variables
- Copy `.env.example` to `.env` and update values as needed.
- Contact form email (Resend):
  - `RESEND_API_KEY` - your Resend API key
  - `CONTACT_TO_EMAIL` - admin email that receives contact messages
  - `CONTACT_FROM_EMAIL` - verified sender in Resend (example: `AutoNest <no-reply@yourdomain.com>`)
